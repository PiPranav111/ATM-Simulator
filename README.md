# 🏧 ATM Simulator (C++)

This is a command-line ATM simulation system built in C++. It mimics real-world ATM functionality including:

- 🔐 User authentication
- 💰 Balance checking
- 💸 Cash withdrawal
- 📱 Mobile number update
- 🔁 Infinite runtime for real-time simulation

## 📁 Project Structure

- `main.cpp` – Main ATM logic
- Written entirely in modern C++ (no external libraries)

## 🧪 How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/ATM-Simulator.git
   cd ATM-Simulator
   ```

2. Compile the code:
   ```bash
   g++ main.cpp -o atm
   ```

3. Run the ATM simulation:
   ```bash
   ./atm
   ```
